Here is interpolation {{ name }}

Just pass a property inside an HTML tag, inside double brackets

This way its only possible to bind a string value:

id="{{myId}}" type="text" value="example-1"

disabled="isDisabled" type="text" value="example-2"

The disabled prop = false, is not being binded:

This way its possible to bind booleans:

[id]="myId" type="text" value="example-2"

Disabled prop = false, being binded:

[disabled]="isDisabled" type="text" value="example-2"
